What is Wasm?

The typical definition you'll find is the following: "WebAssembly, shortened to Wasm, is a binary instruction format for a stack-based virtual machine"

A key point to remember is that just like in our demo there were no instructions that were able to access anything on the system, all it was able to do was manipulate the stack and some memory, this is also the case for WASM. It's entirely sandbox and can't access anything outside of its own environment.

WAT

WAT (yes, that's what it's called) is short for WebAssembly Text Format. While WASM is in fact a binary format, there is a text format for WASM.

WABT

(Aren't these names awesome?) WABT stands for WebAssembly Binary Toolkit. It's a suite of tools used for working with WASM.
